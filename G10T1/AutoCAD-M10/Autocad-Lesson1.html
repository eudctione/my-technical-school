نماد شعاع

R
توضیح: برای نشان دادن اندازه شعاع کمان‌ها و گوشه‌های گرد استفاده می‌شود.

نماد مربع

□
توضیح: برای نشان دادن سطح مقطع مربع در نقشه‌ها استفاده می‌شود.

فرمول‌های محاسباتی

محاسبه مقیاس

مقیاس = اندازه نقشه ÷ اندازه واقعی
توضیح: برای محاسبه نسبت اندازه نقشه به اندازه واقعی قطعه استفاده می‌شود.

محاسبه محیط دایره

C = 2πR = πD
توضیح: برای محاسبه محیط دایره با استفاده از شعاع (R) یا قطر (D).

تبدیل اندازه‌ها

1 اینچ = 25.4 میلی‌متر
توضیح: برای تبدیل اندازه‌ها بین سیستم متریک و سیستم اینچی.
